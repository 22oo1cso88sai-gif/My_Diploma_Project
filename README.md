# Smart Seating System – Exam Seating Management

This project automates the allocation of students to examination halls 
based on room capacity and student information. It helps institutions 
generate seating arrangements efficiently and reduces manual effort.

## Technologies Used
- Python
- Django Framework
- HTML
- CSS
- JavaScript
- SQLite / Database
- Bootstrap (for UI)

## Features
- Student data management
- Automatic seat allocation
- Room assignment based on capacity
- Seating chart generation
- Admin panel for managing students and rooms
- Organized seating output for exams

## Project Structure
- SmartSeating – Core application logic
- SmartSeating_Admins – Admin functionality
- static – CSS, JS, and UI resources
- manage.py – Django project manager

## Purpose
The system is designed to simplify exam hall seating arrangements 
for educational institutions by automating the allocation process.
